## What I Do

- **Framework Architecture** — Design POM-based hybrid frameworks in Java + Selenium + Cucumber BDD, built for scale, maintainability, and parallel execution
- **API Automation** — End-to-end API test suites using REST Assured with schema validation, chained auth flows, and environment-based configs
- **CI/CD Integration** — Jenkins pipelines triggering automated regression on every build, with JIRA-based execution and auto-generated reports
- **Cross-browser & Mobile** — Multi-device validation across Chrome, Firefox, Safari (BrowserStack) and Android/iOS (SeeTest)
- **AI-Augmented Testing** — Generating test scripts and scenarios using GitLab Duo and GitHub Duo, exploring AI agent-based automation
- **Defect Intelligence** — AWS EPS/Gateway log analysis to triage production issues and collaborate on root-cause resolution
